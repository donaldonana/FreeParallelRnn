# ParallelRnn

Pour lancer l'agorithme d'entrainement parrelèle du LSTM

##Parallélé avec Mutex 

1. git clone 
2. cd 
3. cd LSTM/parallel/memoire/
4. ./run_all.sh

On demandera ensuite le nombre de threads (d'unité de traitement) à utiliser

1. git clone 
2. cd

4. ./run_all.sh
